MrBeastify Browser Extension

When I saw the original MrBeastify website I knew I had to make a browser extension that inserts MrBeast into every thumbnail on YouTube.

So here it is!

Add to Firefox
Download for Chrome

How to install (Chrome)

  • Open chrome://extensions
  • Toggle Developer mode on
  • Drag the mrbeastify-1.0.zip file into Chrome
Download for Edge

How to install (Edge)

  • Open edge://extensions
  • Toggle Developer mode on
  • Drag the mrbeastify-1.0.zip file into Edge
